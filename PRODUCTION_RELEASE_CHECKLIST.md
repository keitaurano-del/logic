## Play Console 申請手順

### 1. 製品版 Release Track へビルドアップロード

```
Play Console → Logic → Release → Production
→ "新しいリリースを作成"
→ Android App Bundle (.aab) をアップロード
```

### 2. リリースノート入力

```
【新機能】
- 論理思考を毎日鍛える「Logic」がリリース
- デイリーフェルミ推定問題
- 5つのレッスンカテゴリ

【改善】
- UI一新: 2列グリッド表示対応
- Play Store 決済統一

【既知制限】
- AI問題生成: ベータ版のため
- ロールプレイ: ベータ版のため
```

### 3. ストア情報 最終確認

- [ ] アプリ名: Logic
- [ ] プレビュー: 日本語で正しく表示
- [ ] スクリーンショット: すべてアップロード
- [ ] 説明文: 日本語
- [ ] サポートメール有効
- [ ] プライバシーポリシー URL 有効

### 4. コンテンツレーティング入力

```
Play Console → コンテンツレーティング
暴力: なし
性的内容: なし
18+ コンテンツ: いいえ
→ 保存
```

### 5. ターゲット設定

```
ターゲット国: 日本 + グローバル対応
最小 API レベル: 24 (Android 7.0)
```

### 6. 申請ボタン

```
Play Console → Release → Production
→ "審査のために送信" ボタン
```

**審査期間: 通常 1〜3時間**
