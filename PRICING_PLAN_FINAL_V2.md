**Stripe Web 決済**
- Web ブラウザ上で実装

### 決済方式（将来）
**Google Play Billing**

- **現在: TypeScript スタブのみ（未実装）**

5. localStorage に subscription 情報を保存

## キャンペーン管理

### 期間限定キャンペーン実装方法
```typescript
// subscription.ts
export const PLAN_PRICES = {
  standard_monthly: 390,
  standard_yearly: 2730,
  campaign_yearly: 1980,    // 期間限定
  beta_campaign: 1980,       // 同じ金額
}

// 実装時点で campaign_yearly を期間限定フラグで制御
const CAMPAIGN_END_DATE = new Date('2026-05-31T23:59:59+09:00')
export const isCampaignActive = () => {
  return new Date() < CAMPAIGN_END_DATE
}
```

### キャンペーン終了後
1. `isCampaignActive()` が false を返す
2. Pricing 画面でキャンペーンプランが非表示
3. スタンダード年額 2,730円 のみ表示
4. SVG グラフィック: `pricing_comparison.svg` を使用
