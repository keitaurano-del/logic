## 説明文 (App Store / Play Store 共通・日本語)

### 30字キャッチコピー
論理思考を3分から。AIと一緒に。

### 詳細説明 (4000字以内・改行込み)

```
Logic は、ロジカルシンキング・簿記・プロジェクトマネジメントを「3分から始められる」短いレッスンで体系的に学べる学習アプリです。

■ こんな方におすすめ
・コンサルや外資企業で必要な論理思考を身につけたい
・会議やプレゼンで「で、結論は?」と言われがち
・MECE やロジックツリーを聞いたことはあるが使えない
・簿記3級・2級を試験対策ではなく実務で理解したい
・PMBOK の基礎を短時間でつかみたい

■ 主な機能

【豊富なレッスン】
・ロジカルシンキング: MECE / ロジックツリー / So What・Why So / ピラミッド原則 / 演繹法 / 帰納法 / 形式論理(A→B)
・簿記3級・2級
・プロジェクトマネジメント (PMBOK)

【AI を使った実践練習】
・ロールプレイ: 上司への報告、会議のファシリ、クライアント説明、部下への指示など実務シーンで論理思考を AI 相手に練習
・フェルミ推定: 「日本のコンビニは何店舗?」など、分解思考を AI フィードバックつきで実践
・AI 問題生成: あなた専用の練習問題を AI が生成
・コーヒーブレイク: パートナー関係・買い物・旅行など日常シーンに論理を活かすクイズ

【偏差値 & ランキング】
・8 問のプレイスメントテストで現在地を判定
・全国ユーザーとのランキングで自分の位置を確認
・偏差値に応じた最適な学習ルートを自動提案

【継続のしくみ】
・今日の1問で毎日続けやすい
・連続学習日数 (ストリーク) と完了レッスン数を可視化
・フラッシュカードで弱点を反復学習

【あなたに合わせた表示】
・ライト/ダーク+プレミアムテーマ
・PC・タブレット・スマホすべてに最適化

■ ベータ版について
現在ベータ版として提供中です。ベータ期間中は全機能を無料でご利用いただけます。ご意見・不具合報告はアプリ内「フィードバック」からお願いします。

■ プライバシー
学習データの大半は端末内のみに保存されます。アカウント登録は不要、個人を特定する情報は取得しません。詳しくはプライバシーポリシーをご確認ください。
```

## スクリーンショット撮影リスト

### 必要なサイズ

**iOS (App Store Connect):**
- iPhone 6.7" (1290×2796): iPhone 15 Pro Max など — **必須**
- iPhone 6.5" (1242×2688): iPhone 11 Pro Max など — 任意
- iPhone 5.5" (1242×2208): iPhone 8 Plus など — 任意
- iPad 12.9" (2048×2732): iPad Pro 12.9" — iPad 対応する場合は必須

**Android (Play Console):**
- Phone (1080×1920 以上、最低 2 枚 / 最大 8 枚)
- 7" Tablet (1024×600 以上)
- 10" Tablet (1280×800 以上)

### 撮影する画面 (5 枚構成)

1. **ホーム画面** — ストリーク + 偏差値おすすめカード + 主要 4 カードが見える状態
   - キャプション: 「3 分から始める論理思考トレーニング」

2. **プレイスメントテスト結果** — 偏差値の円 + おすすめレッスンが見える状態
   - キャプション: 「8 問テストで現在地を判定」

3. **ロジカルシンキングのレッスン (MECE 図解付き)** — 図解 step を表示
   - キャプション: 「図解で直感的に理解」

4. **AI ロールプレイのチャット画面** — 上司との会話 + 選択肢 3 つ
   - キャプション: 「AI 相手に実務シーンで実践」

5. **偏差値ランキング** — TOP リストと自分の順位
   - キャプション: 「全国ランキングで切磋琢磨」

### 撮影手順

```bash
# 1. ローカルで dev サーバーを起動 (Capacitor 化前)
cd logic && npm run dev
# → http://localhost:5173 を開く

# 2. ブラウザの DevTools → デバイスツールバー (Cmd+Shift+M)
#    iPhone 14 Pro Max → 1290x2796 設定 → 各画面に遷移してスクショ
#    iPad Pro 12.9 → 2048x2732 設定 → 同様
#    Android 用は Pixel 7 Pro → 1080x2400

# 3. スクショは macOS なら Shift+Cmd+5 → 範囲選択 → 保存
```

撮影前に **placement test を完了** + **2-3 レッスン完了** + **ロールプレイ 1 回プレイ** 状態にしておくと画面に意味のあるデータが入ってベターです。
