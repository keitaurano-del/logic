スーパーの商品カテゴリを「食品・日用品・その他」に分けた。この分け方はMECEといえるか？
A
いえる。3つに分かれているから
B
いえない。「その他」が残るためダブりや漏れが生じやすい
C
条件次第でどちらにもなりえる
D
MECEの概念は分け方の数に依存しない
正解は B
「その他」は「何でも入る箱」。MECEは各カテゴリが相互排他的かつ全体網羅的であることが条件です。「その他」があると全体網羅はできても相互排他が崩れます。
