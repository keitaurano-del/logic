- BETA_MODE = true（全機能無料）

- Android = 将来Google Play Billing

### Duolingo参考

| Duolingo | Logic（提案） |
|----------|-------------|
| 無料+広告 | 無料（広告なし、機能制限） |
| Super ¥1,990/月 | Standard ¥480/月 |

**ポイント: Duolingoより大幅に安く設定。** 教育アプリの価格感度は高い。最初は安くしてユーザー基盤を作り、価値が証明されてから段階的に値上げ。

### 有料化タイミング

```
ベータ期（今）     → 全機能無料、フィードバック収集
ユーザー100人突破  → Free/Standardの2プラン導入
ユーザー1,000人突破 → Premium追加、年額プラン導入
```

### 実装ステップ
1. Free制限の実装（フェルミ1問/日、基本レッスンのみ）
2. Web版: Stripe Checkout（既存コード活用）
3. Android版: Google Play Billing（`@capawesome-team/capacitor-android-billing`）
4. バックエンド: Supabase `subscriptions` テーブルで統合管理

### 収集チャネル（優先順）

| チャネル | 導入コスト | 効果 | 実装難度 |
|---------|----------|------|---------|
| **①アプリ内フィードバックフォーム** | ¥0 | ◎ | 既存（FeedbackScreen.tsx） |
| **②Play Storeレビュー** | ¥0 | ◎ | 自動 |
| **③X(Twitter) #Logic フィードバック** | ¥0 | ○ | 監視スクリプト |
| **④メール（genspark.email）** | ¥0 | ○ | 既存 |
| **⑤LINEオープンチャット（ベータコミュニティ）** | ¥0 | ◎ | 開設のみ |

### フィードバック → Jira 自動化パイプライン

```
[アプリ内フォーム]
     ↓ submit
[Supabase `feedback` テーブル]
     ↓ Edge Function (webhook)
[Jira API → チケット自動起票]
     ↓
[Slackに通知]
```

**実装手順:**
1. Supabase に `feedback` テーブル作成（user_id, type, message, screen, app_version, created_at）
2. FeedbackScreen.tsx を Supabase書き込みに変更（現在はメール送信のみ？）
3. Supabase Edge Function: INSERT → Jira REST API → チケット起票
4. Jiraラベル自動付与: feedback + user-reported / feature-request / ux-improvement

### フィードバック分類・対応ルール

| 分類 | 対応SLA | Jira Issue Type |
|------|---------|----------------|
| クラッシュ/データ消失 | 24時間以内 | バグ (P0) |
| UI不具合 | 1週間以内 | バグ (P1) |
| 機能要望（多数票） | 次スプリント検討 | Feature |
| UX改善 | バックログ | ストーリー |

| ドメイン | gensparkclaw.com（既存） | ¥0 |
| **合計** | | **¥0-1,000/月** |

### スケール時のコスト見積もり

| ユーザー数 | Render | Supabase | その他 | 合計 |
|-----------|--------|----------|--------|------|
| 〜100 | Free | Free | — | ¥0 |
| 100-1,000 | Starter ¥1,100 | Free | Sentry Free | ¥1,100 |
| 1,000-5,000 | Standard ¥3,800 | Pro ¥3,800 | Sentry ¥3,800 | ¥11,400 |
| 5,000-10,000 | Pro ¥8,500 | Pro ¥3,800 | 各種 ¥5,000 | ¥17,300 |

### コスト最適化ポイント
- **RenderのFreeプラン**: スリープあり（15分無操作で停止）。ユーザー増えたらStarterに上げる
- **Supabase Free**: 500MB DB / 50,000 Auth MAU → 当面十分
- **画像/アセット**: Cloudflare R2（10GB無料）で配信コスト削減
- **AI API**: OpenAI/Anthropic呼び出しはサーバーサイドで、使用量に応じて課金。フリーミアム制限で抑える

### 損益分岐点

```
月額固定費: ¥11,400（1,000ユーザー時）
Standard ¥480 × 転換率3% × 1,000人 = ¥14,400/月
→ 1,000ユーザーで黒字化可能
```
