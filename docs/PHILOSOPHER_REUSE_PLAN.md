# 哲学者イラスト再活用計画 (SCRUM-131)

作成: 2026-04-26  
ステータス: Keita-san 承認待ち

---

## 背景

ランク廃止（SCRUM-130）により、哲学者キャラクター（10体）はUIから除外された。
イラスト資産として以下の再活用案を提案する。

---

## 活用案

### 案A: ロールプレイの対話相手キャラ（最推奨）

**概要**
哲学カテゴリのレッスン完了後または単独のロールプレイ画面で、
哲学者キャラクターと「問答形式」で対話するモード。

**対象キャラクターとテーマ**

| キャラクター | 対話テーマ |
|---|---|
| ソクラテス | 問答法・無知の知・定義を崩す訓練 |
| プラトン | イデア・認識の本質・洞窟の比喩 |
| アリストテレス | 論理学・三段論法・帰納と演繹 |
| デカルト | 方法的懐疑・確実なものを積み上げる |
| カント | 義務と規則・普遍的法則 |
| ニーチェ | 前提を壊す・常識を疑う |

**実装イメージ**
- RolePlayScreen に「哲学者と対話する」モードを追加
- キャラクターカード選択（既存のシナリオ選択UIと同形式）
- AIがキャラクターの口調・思想で応答
- レッスン77〜81（哲学カテゴリ）修了後にアンロック

**工数見積もり**: 中（AIプロンプト設計 + UI接続）

---

### 案B: 哲学カテゴリのヘッダー装飾（低コスト）

**概要**
RoadmapScreen の「哲学・思考の原理」カテゴリヘッダーに
代表キャラクター（ソクラテス）のイラストを配置。

**実装イメージ**
- カテゴリヘッダーの右側にイラストをサムネイル表示
- タップするとキャラクター紹介モーダルが開く

**工数見積もり**: 低（CSS配置のみ）

---

### 案C: 節目レベルアップの演出

**概要**
Lv10 / 25 / 50 / 99 などの節目でキャラクターが登場して
名言とコメントを表示するポップアップ。

**例: Lv10 到達時**
> デカルトのイラスト +「方法的懐疑の精神が身についてきた。」

**工数見積もり**: 中（LevelUpModal 新規作成）

---

## 推奨実装順

1. **案B**（即実装可能・低コスト）
2. **案A**（哲学カテゴリ完成後に並行実装）
3. **案C**（レベルシステム安定後）
