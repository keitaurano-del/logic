# Logic v2 リデザイン設計書
> 参照: Awarefy動画分析 (2026-04-26) + Keita-san指示「A/B/C全部やる」

## 設計方針

### コアコンセプト
**「論理思考の習慣化アプリ」** — Awarefyが「感情との向き合い」を習慣化するように、Logicは「論理的思考」を毎日の習慣にする。

### デザイン言語（変更なし・強化）
- カラー: 現状の `#3B5BDB` ネイビーブルーを維持（変えない）
- 背景: `#F0F4FF` → `#FFFFFF` or `#F8FAFF` に変更（より清潔感）
- カード: border-radius 14px → **20px** に統一
- 余白: カード間gap 12px → **16px**
- タップフィードバック: `transform: scale(0.97)` + `transition: 120ms ease`
- フォント: Inter Tight 継続

## B. 画面構成・情報設計刷新

### B-1. BottomNav（4タブ → 変更なし、アイコン刷新）
```
ホーム | レッスン | 記録 | プロフィール
```
アイコンをより「まるみのある」スタイルに統一。アクティブ状態は塗りつぶし（現状と同じだが視覚的に強調）

### B-2. ホーム画面 全面刷新

**現状の問題点**:
- 情報が縦に詰まりすぎ
- 「今日やること」が不明確
- カードデザインがバラバラ

**v2 構成**（上から順）:
```
[ナビバー] Logo + ストリーク + アバター

[今日のルーティン] ← NEW（Awafyのチェックイン相当）
  朝のチェックイン: 「今日の問題を解く」
  └ DailyFermi カード（大きなCTA）

[今週の進捗] ← コンパクト化
  週カレンダー（月〜日の丸印）+ ストリーク数

[続きから] ← NEW
  前回中断したレッスン or 次のおすすめ1件
  （水平スクロール不要、1枚カードで十分）

[学習コース] ← 水平スクロール化
  ┌─────────┐ ┌─────────┐ ┌─────────┐
  │ロジカル  │ │ケース面接│ │哲学     │
  │5レッスン │ │4レッスン │ │5レッスン│
  └─────────┘ └─────────┘ └─────────┘
  （Awarefyのコース水平スクロールに相当）

[AI練習メニュー] ← 2カラムグリッド
  ┌───────────┐ ┌───────────┐
  │AI問題生成  │ │ロールプレイ│
  └───────────┘ └───────────┘

[Lv / XPカード] ← 現状維持・デザイン洗練
```

### B-3. レッスン画面（RoadmapScreen）刷新

**現状**: カテゴリ別にレッスンをリスト表示

**v2**:
- カテゴリカードを大きく（Awarefyのコース一覧に近い）
- 各カテゴリに「所要時間」「レッスン数」「難易度」を明示
- タップでカテゴリ内レッスン一覧に展開（アコーディオン or 別画面）
- 完了レッスンに「チェックマーク」、未完に「●」で進捗可視化

### B-4. 記録タブ（StatsScreen）刷新

**現状**: 概要/履歴/分析の3タブ

**v2**（Awarefyのタイムライン参照）:
- **月カレンダービュー**: 学習した日をドット表示（塗り/空で視覚化）
- **週次フィルター**: 日/週/月 切り替え（シンプルなセグメントコントロール）
- **学習ログ**: 日付ごとにやったこと（レッスン名 + 所要時間 + 獲得XP）
- AIウィークリーレポートカード（週1回「あなたの今週の学習」まとめ）← NEW

### B-4（続き）. 記録タブ 詳細設計

**上部: 月カレンダー**
```
    4月
 Mo Tu We Th Fr Sa Su
  ●  ●  ○  ●  ●  ○  ○   ← 塗り=学習済, 空=未学習
  ○  ●  ●  ●  ○  ○  ○
  ...
```
- 今日はリング強調
- 当月ストリーク数・総学習日数をカレンダー下に表示

**中部: 期間フィルター**

各ログをタップするとレッスン詳細へ


### B-5. プロフィール画面 刷新設計

**現状の問題点**: 情報が縦にただ並んでいるだけ、ヒーローエリアが弱い

**v2 構成**:
```
[ヒーローエリア] 濃いブルー背景
  アバター（イニシャル円）
  ユーザー名
  Lv.XX  ████████░░  (XPバー)
  「次のレベルまであとXXXXXP」

[実績グリッド 3列]
  連続学習   完了レッスン   総XP
  14日       23個          1,250

[学習サマリーカード]
  今週: 5日学習 / 3レッスン / +180XP
  （ミニカレンダーつき）

[設定リスト]
  アカウント   →
  通知設定     →
  プラン       現在: Free  →
  フィードバック →
  ログアウト
```

**ヒーローエリアの変更ポイント**:
- 背景を `linear-gradient(160deg, #1E2D6B, #3B5BDB)` に
- XPバーを太く（12px）して進捗感を出す
- アバターに薄いglow効果

## C. 学習コース設計刷新

### C-1. レッスン所要時間の明示

全レッスンに所要時間ラベルを付与（現状一部のみ）:
- 1〜3分: クイズ中心のレッスン
- 3〜5分: 標準レッスン
- 5〜10分: 詳細解説レッスン

RoadmapScreen・HomeScreenのカード上に常に表示。

### C-2. オンボーディング刷新（SCRUM-118相当の再設計）

**現状**: 4ステップ（価値説明→機能紹介→プラン→開始）

**v2 フロー**（Awarefy参照）:
```
Step 1: コンセプト訴求（「論理思考を毎日の習慣に」）
Step 2: 目的選択 ← NEW
  □ 仕事での論理思考を鍛えたい
  □ 就活・面接対策がしたい
  □ 日常の思考を整理したい
  □ 純粋に知的好奇心から
Step 3: 最初のリマインダー設定 ← NEW
  「毎日何時に学習しますか？」
  → 通知許可取得もここで
Step 4: プラン選択（現状維持）
Step 5: 「まず1レッスンやってみよう」← NEW
  → 目的に応じたおすすめレッスンを自動選択して直通
```

### C-3. 「今日のルーティン」概念の導入

Awarefyの「チェックイン/チェックアウト」相当:
- **朝のルーティン**: 今日のDailyFermi（1問）を解く
- **夜のルーティン**: 1レッスン（3〜5分）を完了する

ホーム上部に「今日のルーティン」カードとして常時表示。完了したらチェック。

### C-4. 学習パス（ラーニングパス）の明確化

現状のRoadmapは「全レッスン一覧」になっている。v2では:
- **「入門コース」**: 初心者向け6レッスン固定パス（2週間で完走想定）
- **「ビジネス強化コース」**: ケース面接 + 提案・伝える技術
- **「哲学・深掘りコース」**: 哲学 + ロールプレイ
- 各コースに「〇〇日で完了」の目安表示
