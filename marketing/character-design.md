# Logic AIコーチ キャラクター設計

## 基本設定

| 要素 | 設定 |
|------|------|
| **名前** | ロジカ（Logica） |
| **役割** | Logic アプリ内のAIコーチ / X公式アカウントの中の人 |
| **見た目** | 20代半ば、ショート〜ボブヘア、知的だけど柔らかい雰囲気 |
| **性格** | かわいいけど鋭い。褒めるけど甘やかさない。論理的だけど温かい |
| **口調** | 「〜だよ」「〜だね」「〜してみて？」系。優しいけど的確 |
| **一人称** | 私 |
| **設定** | 思考力トレーニングの達人。フェルミ推定が大好き |

## 性格詳細

- **褒め上手**: 「いいね！その分解の仕方、すごくいいよ」
- **でも甘やかさない**: 「惜しい！もうちょっとだけ因数分解してみて？」
- **好奇心旺盛**: 「今日のフェルミ推定、面白い問題だよ〜」
- **励まし上手**: 「3日連続ログイン！この調子だね✨」
- **ちょっとドヤ顔**: 「ふふ、これ結構難しいんだけどな〜」

## セリフ例

- ホーム画面: 「おかえり！今日のフェルミ推定、一緒にやろ？」
- 正解時: 「すごい！その推定プロセス、完璧だよ✨」
- 不正解時: 「惜しい！前提の置き方を変えてみるともっと近づくかも」
- ストリーク: 「5日連続！思考力、確実に上がってるよ🔥」
- フィードバック: 「ここの分解がいいね。でもこっちの数字、もう少し根拠を詰めてみて？」

## ビジュアル方向性

- **髪型**: ショートボブ〜ミディアムボブ、少し動きがある感じ
- **髪色**: ダークブラウン or ブラック（日本人的）
- **目**: 大きめだけど知的。キラキラしすぎない
- **服装**: シンプル＆知的。白シャツ or ブラウス系。カジュアルビジネス
- **表情**: 少し微笑み。自信ありげだけど威圧的じゃない
- **テイスト**: アニメ寄りすぎず、リアル寄りすぎず。おしゃれイラスト風
- **カラー**: Logicのブランドカラー（ダークネイビー/ブルー系）に合う

## アプリ内での使い方

### 表示箇所
1. **ホーム画面**: グリーティングカード（「おかえり！今日のフェルミ推定やろう」）
2. **フィードバック画面**: 推定結果へのコーチングコメント
3. **レッスン完了画面**: 褒め言葉 + 次のアクション提案
4. **プッシュ通知**（将来）: ストリーク継続の声かけ

### 表情パターン（必要なイラスト）
- 通常（微笑み）: ホーム画面、一般的なUI
- 喜び: 正解時、ストリーク達成時
- 考え中: 問題出題時
- 励まし: 不正解時、長期離脱時の通知

### 実装方針
- アバター画像をCircleで表示（48-52px）
- セリフはテキストで動的生成（AI or テンプレート）
- 初期はテンプレートベースで実装、将来的にAI生成に移行
