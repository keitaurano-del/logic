---

## 各CXOに追加するファイル

```
agents/{cxo}/
├── AGENT.md        ← 役割定義（既存）
├── KNOWLEDGE.md    ← 蓄積された知識（既存）
├── RULES.md        ← 行動ルール（既存）
├── DECISIONS.md    ← 判断ログ（新規）
└── ACHIEVEMENTS.md ← 成果ログ（新規）
```

### DECISIONS.md の書式
```markdown
## YYYY-MM-DD: [判断の要約]
- **状況**: 何が起きていたか
- **選択肢**: 何と何を比較したか
- **判断**: 何を選んだか
- **根拠**: なぜそう判断したか
- **結果**: どうなったか（事後に追記）
```

### ACHIEVEMENTS.md の書式
```markdown
## YYYY-MM-DD: [成果の要約]
- **内容**: 何を達成したか
- **インパクト**: どんな影響があったか
- **学び**: 次に活かせること
```

---

## 成長ルール

### ナレッジの更新タイミング
- タスク完了時: 学んだことをKNOWLEDGE.mdに追記
- 判断時: DECISIONS.mdに記録
- 週次: その週の成果をACHIEVEMENTS.mdに追記

### CEOの責任
- 週次でCXO_GROWTH.mdのナレッジ量テーブルを更新
- 成長が止まっているCXOには追加のタスクを振る
- ナレッジが増えていないCXOは「仕事をしていない」と判断する
