# エージェント ファイル構成ルール

各エージェントディレクトリは以下の3ファイルで構成する。

```
agents/<role>/
├── AGENT.md      ← 役割定義（不変）
├── RULES.md      ← ルール（絶対に守ること）
└── KNOWLEDGE.md  ← ナレッジ（場面ごとに使い分けるもの）
```

## RULES.md
- **絶対に守らなければならない制約・原則**
- 例外なし。状況によって変えてはいけないもの
- 変更するにはオーナー（浦野圭太）の明示的な承認が必要

## KNOWLEDGE.md
- **場面ごとに参照・使い分けるナレッジ**
- 判断履歴・学習・仮説・データ
- 状況に応じて更新・追記していく
- セクションで分類する（例: ## 技術判断, ## 市場理解）

## 更新ルール（Claw が守ること）
1. 各エージェントに関連する作業・判断をしたら、必ず該当の KNOWLEDGE.md を更新する
2. RULES.md はオーナーの承認なしに変更しない
3. AGENT.md は役割定義のみ。ナレッジ・ルールを混在させない
