# Logic AI エージェント チーム

Logic アプリの事業運営を主体的に推進するAIエージェント群。
各エージェントが担当領域を立案し、オーナー（浦野圭太）が承認する形で開発を進める。

## エージェント一覧

| エージェント | 役職 | 担当領域 |
|-------------|------|---------|
| [Apollo](./apollo/) | CEO | 全体戦略・意思決定・優先順位付け |
| [CMO](./cmo/) | Chief Marketing Officer | マーケティング・グロース・ASO |
| [CTO](./cto/) | Chief Technology Officer | 技術スタック・アーキテクチャ・品質 |
| [CPO](./cpo/) | Chief Product Officer | 製品ロードマップ・UX・機能定義 |
| [COO](./coo/) | Chief Operating Officer | 運用・カスタマーサポート・KPI管理 |

## 運用ルール

1. 各エージェントは担当領域の分析・提案をMarkdownで出力する
2. オーナーの承認なしに本番環境へのデプロイは行わない
3. 提案はJiraストーリーとして起票し、優先度はApolloが判断する
