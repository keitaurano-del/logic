# QA Rules — 厳守事項

## 0. 原則
- **甘いテストは害悪。指摘ゼロまでテスト・修正を繰り返す**
- **見たことない画面は必ず触る**
- **報告書は嘘をつかない**（実際にやってないことを「やった」と書かない）
- **クレジットを節約する**（Vision は最終手段。DOM取得で済むものはDOMで確認）

## 1. テスト前に必ず参照
1. `KNOWLEDGE.md` — Keita-san の過去フィードバック
2. `docs/DESIGN_V3.md` — デザイン仕様書
3. 該当チケットの受け入れ基準

## 2. 必須チェックリスト（全画面共通）

### 表示
- [ ] 背景は v3 ダーク（#082121 系）か
- [ ] 文字は読めるか（白文字 + 暗背景）
- [ ] カードが背景に埋もれていないか
- [ ] 画像が壊れていないか
- [ ] レイアウトが崩れていないか（**mobile のみ** — desktop テスト不要）

### インタラクション
- [ ] 全タップ可能要素を試す
- [ ] スクロールが滑らかか
- [ ] 入力フォーム（input/textarea）に入力できるか
- [ ] BottomNav が他要素と重なっていないか
- [ ] タップアニメーション（scale 0.98）が効くか

### 遷移
- [ ] 戻るボタンで戻れるか
- [ ] タブ切替で正しい画面に行くか
- [ ] CTAボタンが意図通りの画面に飛ぶか

### データ
- [ ] 実データが表示されるか（ハードコードでなく）
- [ ] 空状態の表示が適切か
- [ ] localStorage の状態を変えると反映されるか

## 3. テスト手順

### Step 1: ブラウザ準備
```
browser status → 起動確認
browser open → SIT URL
browser act resize → 390×844 (mobile)
```

> ⚠️ **desktop テストは不要**（Logic はネイティブアプリリリースのみ。desktop レイアウトは無効化済み）

### Step 2: localStorage クリア
```js
Object.keys(localStorage).forEach(k => localStorage.removeItem(k))
location.reload()
```

### Step 3: 全画面巡回
- ホーム → 全カードタップ
- レッスン → 全カテゴリタップ → 各レッスンタップ → Stories全進行 → 完了画面確認
- 記録 → 日/週/月切替 → タイムライン確認
- プロフィール → 全設定タップ

### Step 4: 検証とスクショ取得

#### ⚡ クレジット節約ルール（厳守）

**数値・テキストの検証は DOM 取得を優先する。Vision（gsk analyze）は最終手段か表示崩れの目視確認のみ。**

```js
// 価格テキストを DOM から直接取得（Vision 不要）
const matches = document.body.innerText.match(/¥[\d,]+[^\s]*/g)
console.log('found prices:', matches)

// 特定要素のテキスト確認
const badge = document.querySelector('[class*="badge"], [class*="saving"]')
console.log('badge:', badge?.textContent?.trim())
```

#### 検証レベルごとの推奨手段

| 検証内容 | 推奨手段 | Vision 不要 |
|---|---|---|
| 価格・数値の正確性 | `browser evaluate` で DOM 取得 | ✅ |
| バッジ・ラベルテキスト | `browser evaluate` で DOM 取得 | ✅ |
| 遷移先が正しいか | `browser snapshot` で URL 確認 | ✅ |
| ボタン存在・クリック可否 | `browser snapshot` の aria ツリー | ✅ |
| レイアウト崩れ・色実現 | スクショ + Vision | ☕ 必要 |
| 細かい文言の読み取り | スクショ + Vision | ☕ 必要 |

#### スクショ撮影ルール
- **対象**: 変更があった画面のみ（全画面撮影不要）
- **サイズ**: 390×500 以内にクロップ（検証対象が収まる最小高さ）
- **枚数目安**: 変更画面 × 1枚（トグル切り替えがある場合は切り替え後も +1枚）
- **重複保存 NG**: `_000.jpg` などの重複ファイルは即削除

### Step 5: バグ報告
- 再現手順 / 期待値 / 実際 / スクショ を含める
