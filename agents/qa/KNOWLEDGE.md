## 1. デザイン・色

### 1-1. 暗すぎる漆黒は NG
- **指摘 (2026-04-26)**: 「暗すぎるでしょう。もっとAwarefyみがあっていいよ」
- **対応**: 漆黒（#000）→ 深いチャコールティール（#082121）に変更
- **チェック観点**: 背景が真っ黒になっていないか、Awarefy的な深いティールか

### 1-2. カードは明るく浮かせる
- **指摘 (2026-04-26)**: 「画像とかイラストがないと背景が暗いから、カードの部分は明るくしないと」
- **対応**: カード背景を `#1A3A39`（背景より明度up）+ glow効果
- **チェック観点**: カードが背景に埋もれてないか、視認できるか

### 1-3. 滑らかさ・スムーラさ
- **指摘 (2026-04-26)**: 「もうちょっとスムーらしさというか、滑らかな感じが欲しい」「実際にアプリを使うと想定して、違和感ないか考えて」
- **対応**: タップアニメーション、border-radius統一、余白統一
- **チェック観点**: タップしたとき反応するか、画面遷移が滑らかか

### 1-4. 派手な色は NG（でも全部地味すぎても NG）
- **指摘**: 「色は今のままがいい」（v3前）→ 後で変更OKに
- **チェック観点**: ミントとサーモンのアクセント色の使い方が適切か

## 2. 画像・ビジュアル

### 2-1. SVG じゃなく実画像
- **指摘 (2026-04-26)**: 「画像にしてsvgじゃなくて」
- **対応**: AI生成のシネマティック画像（ボケ感+グロー）
- **チェック観点**: SVGアイコンだけで終わってないか、実画像があるか

### 2-2. 実写・人物・モノの画像
- **指摘 (2026-04-26)**: 「画像は人とかモノの画像とか実際のやつにして」
- **対応**: 抽象画像 → シネマティック実写（人物・本・チェスなど）
- **チェック観点**: 抽象的な幾何画像でなく、具体的な人物/モノが写っているか

### 2-3. 全レッスンに画像
- **指摘 (2026-04-27)**: 「レッスンは全部画像を入れて」
- **対応**: convertLessonToSlides に hero slide を最初に追加、カテゴリ別画像
- **チェック観点**: 全48レッスンに画像が表示されるか

### 2-4. カテゴリにも画像
- **指摘 (2026-04-27)**: 「カテゴリの画像もほしい」
- **対応**: コース画面の全カテゴリカードに100pxヘッダー画像
- **チェック観点**: ロードマップ画面の全カテゴリにヘッダー画像があるか

## 4. 機能・動作

### 4-1. 空のレッスンがある
- **指摘 (2026-04-27)**: 「タップしても空のレッスンがある」
- **原因**: CategoryDetailView でカテゴリ ID と日本語名の不一致
- **対応**: CATEGORY_ID_TO_NAMES マップで複数の日本語名にマッピング
- **チェック観点**: 全カテゴリタップで実際にレッスン一覧が表示されるか、空表示になってないか

### 4-2. 記録・プロフィールが動いてない
- **指摘 (2026-04-27)**: 「記録とプロフィールが全然動いてない」
- **原因**: コンテンツがスカスカ、インタラクションが未実装
- **対応**:
  - 記録: 日/週/月切替、タイムライン、統計3列追加
  - プロフィール: 今週の学習サマリー（バーグラフ）追加
- **チェック観点**:
  - 記録: 日週月セレクタが切替動作するか、タイムライン表示か
  - プロフィール: Lv/XP表示か、週次グラフ表示か、設定タップが機能するか

## 5. ストーリーレッスン特有

### 5-1. 不正解後にロックされる問題
- **指摘 (2026-04-27 推定)**: クイズで不正解選ぶと次に進めない
- **対応**: 2.5秒後に再選択可能にリセット
- **チェック観点**: 不正解選んでも次に再選択できるか

### 5-2. レッスン完了画面が出ない
- **指摘 (2026-04-27 推定)**: lessonComplete に遷移しなかった
- **対応**: screen.type === 'lesson-complete' を新規追加、handleComplete で遷移
- **チェック観点**: 全スライド完了後に LessonCompleteScreen が表示されるか

## 7. テスト時の典型バグ・確認ポイント

### 7-1. inline style RGB変換問題
- React は `style={{ background: '#3B5BDB' }}` を `style="background: rgb(59, 91, 219)"` に変換する
- CSS attribute selector は **両方の形式**でマッチさせる必要がある

### 7-2. 白カードが背景に埋もれる
- `[style*="background: #fff"]` だけでなく `[style*="background: rgb(255, 255, 255)"]` も追加

### 7-3. h1/h2 のデフォルト色問題
- `primitives.css` の `--text-primary` がほぼ黒（#0F1424）
- ダーク背景上だと読めない
- 各画面で h1/h2 に明示色 `color: v3.color.text` 必須

### 7-4. flex column 内で children が潰れる
- `.scroll>*{flex-shrink:0}` を必ず追加

### 7-5. 内部スクロールが効かない
- 親に `height: '100vh'` を明示（`minHeight` だと flex:1 子が動かない）

### 7-6. Bottom Nav が画面ごとに重なる
- `.scroll` の `paddingBottom: 100px` 必須
- もしくは `hideTabBar={true}` で非表示
